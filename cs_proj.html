Projects

BMP Stenography

Summary: Developed a versatile steganography tool in C that conceals and reveals text within 24-bit BMP image files. The application features intricate bit manipulation to store text in the least significant bits of the color channels, allowing hidden communication. Implemented error handling to support only specific file formats and integrated functionality for information retrieval, message revealing, and text hiding.

Malloc

Summary: I spearheaded the creation of a cutting-edge memory allocator, simulating malloc and free operations without built-in utilities. I started with a basic model and strategically integrated block splitting for better efficiency. The game-changer was the implementation of a doubly linked free list, resulting in a significant performance leap. Through rigorous testing and precision in low-level C programming, I not only enhanced my expertise in memory management but also showcased innovation in systems programming.

Shell Project

Summary: Developed a dynamic shell application, integrating a robust command mechanism and an innovative plugin system. Demonstrated expertise in command parsing, error handling, and optimized use of classic functions like strtok and strsep for versatile string operations. Rigorously tested for peak performance in intricate Linux environments, this initiative showcases my advanced computational skills and dedication to setting pioneering industry benchmarks.

Autocomplete

Summary: I developed an efficient autocomplete search engine that taps into the advanced capabilities of the De La Briandais Trie for precise word prediction. This engine is complemented by a seamlessly integrated predefined dictionary, ensuring users receive accurate and relevant word suggestions. Additionally, I designed an intuitive user interface, allowing for dynamic word completions and featuring simple commands for both word finalization and program termination."
